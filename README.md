# Headless KDE + NoMachine Setup (Xorg Dummy)

## Features
- KDE Plasma Desktop (X11)
- NoMachine Remote Desktop
- Xorg Dummy Display
- Resolutions:
  - 1920x1080
  - 1600x1000
  - 1600x900
- Optional Tailscale VPN

## Install

```bash
git clone https://github.com/yourname/headless-kde-nomachine
cd headless-kde-nomachine
chmod +x install.sh
./install.sh
